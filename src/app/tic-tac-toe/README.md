# Tic Tac Toe 구현 연습

## 목표

- [React 튜토리얼 사이트](https://react.dev/learn/tutorial-tic-tac-toe)에 있는 Tic Tac Toe 게임을 직접 구현

## 작업 계획

1. UI 구현
    - 로직 부분을 모두 제외한, 게임에 필요한 UI를 먼저 구현
    - 구현의 편의를 위해 flex 요소를 적극 활용
2. 승패를 포함한 기본 게임 로직 구현
3. 게임 기록 목록 구현
4. 특정 기록으로 돌아가기 구현
5. 코드 정리

## 로직 구현 정리

### 게임 진행 관련

#### 상태값

- x, y 좌표 별 선택한 플레이어 정보를 가지고 있는 board
- 현재 진행 중인 플레이어
- 승리 여부
